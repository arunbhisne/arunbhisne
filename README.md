# Arun Bhisne — Product Designer, Agentic Experiences

I design the interaction contract between humans and semi-autonomous AI agents — the trust signals, failure states, intervention points, and orchestration patterns that make agentic systems feel controllable rather than opaque.

**Current focus areas:**
- **Orchestration & Handoff** — what passes between agents, and when humans need to step in
- **Confidence & Uncertainty** — making probabilistic outputs legible without overwhelming the user
- **Proactive vs. Confirmatory** — defining the boundary between agent autonomy and user consent
- **Failure & Recovery Design** — probabilistic output handling, graceful degradation, course correction
